# 💰 Expense Tracker - Трекер прочитанных книг

**Автор:** Деркач Лев Алексеевич
**GitHub:** https://github.com/derkachlev/Book-Tracker.git

## 📖 Описание

Book Tracker — это графическое приложение на Python с использованием библиотеки Tkinter для отслеживания личных расходов. Программа позволяет добавлять расходы, фильтровать их по категории и дате, рассчитывать сумму за выбранный период, а также сохранять данные в JSON-файл.

## 🚀 Функциональность

- ✅ Добавление расходов (сумма, категория, дата)
- ✅ Просмотр всех расходов в таблице
- ✅ Фильтрация расходов по категории
- ✅ Фильтрация расходов по дате
- ✅ Подсчёт суммы расходов за выбранный период
- ✅ Удаление выбранных записей
- ✅ Автоматическое сохранение в JSON
- ✅ Загрузка данных при запуске
- ✅ Валидация ввода (положительная сумма, формат даты)

## 🛠 Установка и запуск

### Требования
- Python 3.6 или выше
- Стандартные библиотеки: `tkinter`, `json`, `os`, `datetime`
